# CryptoMarket

CryptoMarket is a mobile app built with Expo and React Native that tracks crypto markets, surfaces news, and provides a simple wallet view. It includes Firebase authentication with email/password and Google sign-in plus password reset.

## Features

- Crypto market listings and details
- News feed with article detail pages
- Wallet overview
- Firebase auth (email/password)
- Google signup and password reset

## Tech stack

- Expo + React Native
- Expo Router
- Firebase
- Axios

## Getting started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the app:

   ```bash
   npm run start
   ```

## Scripts

- `npm run start`
- `npm run android`
- `npm run ios`
- `npm run web`
- `npm run lint`

## Configuration

- Firebase config lives in `src/config/firebase.js`. Make sure it matches your Firebase project.
